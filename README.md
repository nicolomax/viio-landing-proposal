# VIIO Landing – New Proposal

Nueva propuesta de landing para VIIO desarrollada como exploración visual desacoplada del sistema actual.

## Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Enfoque
Arquitectura modular basada en componentes (Hero, Navbar, secciones) alineada a Figma, pensada para iteración y posible integración futura.

## Estado actual
- Desktop: funcional
- Mobile: en desarrollo (puede presentar inconsistencias)

## Consideraciones
- No está alineado aún con la arquitectura actual (Next.js / Vercel)
- No incluye lógica de producto ni integración con datos reales
- Requiere refinamiento de componentes y criterios de desarrollo para producción
